# TcGM.sf2

GS(MSGS)互換の GM1 音源配列を持つ、**正弦波加算合成方式**のサウンドフォントです。
録音サンプルは一切使用せず、すべての音色を正弦波の重ね合わせだけで合成しています。

## 試聴

[▶ YouTube で試聴する](https://youtu.be/pV7XDJMSUX4)

## 特徴

- **GM1 128音色 + 標準GMドラムキット**をGS(MSGS)互換のバンク/プログラム配列で収録
  (Bank 0 = メロディ楽器、Bank 128 / Preset 0 = ドラムキット)
- 音源方式は**完全な加算合成**(正弦波の足し合わせのみ)。打楽器やブレスノイズ系の音色も、
  多数の正弦波クラスターでノイズ的な質感を近似しています
- 各音色の実用音域について**鍵盤1つにつき1音を個別合成**(音域外の端の鍵盤はピッチ追従で延長)
- フィルター(ローパス+レゾナンス)とLFOによる音色の時間変化を追加し、
  特にシンセ系の音色が単調にならないよう調整
- pan・reverb send・chorus send を音色ごとに設定。SF2の標準モジュレータにより
  CC1(モジュレーションホイール→ビブラート)・CC7/CC11(音量/エクスプレッション)・
  CC10(パン)・CC91/CC93(リバーブ/コーラス送り)・ピッチベンドに対応

## ファイル

- `TcGM.sf2` — 完成したサウンドフォント本体(約490MB、Git LFS管理)
- `scripts/` — 生成に使用したPythonビルドパイプライン
  - `synth_engine.py` — 加算合成エンジン(倍音・非調和性・ノイズクラスター・ループポイント検出)
  - `gm_data.py` / `timbre_recipes.py` — GM1 128音色のカテゴリ別音色レシピ
  - `drum_recipes.py` — 標準GMドラムキットの合成レシピ
  - `sf2_writer.py` — 自作のSF2(RIFF)バイナリライター
  - `build_sf2.py` — 全体のビルドスクリプト

## 使い方

VirtualMIDISynthやFluidSynthなど、SF2に対応したMIDIシンセ/プレイヤーで読み込んでご利用ください。
Git LFSを使用しているため、`git clone` する場合は事前に `git lfs install` が必要です。

## 制限事項

- 実機のGS音源や録音サンプル音源を忠実に再現したものではなく、**加算合成による近似音色セット**です
- SF2フォーマット自体に「ディレイ」専用のジェネレータは存在しないため、ディレイエフェクトは
  コーラス送りによる近似にとどまります
